Ryan Hueckel

I am a game programmer currently finishing up my degree in Computer Science: Game Design at University of California: Santa Cruz. In college I have collaborated in groups made up of 3-25 people to create games in many different styles/genres such as co-op 2D platformers, 3rd person shooters, and isometric action-adventure games. My favorite things to code are player controllers, AI, and UI. I love making games feel as satisfying to play as possible.

Below is a collection of some of the projects I have worked on!

Ensol

Ensol is an isometric action-adventure game where you uncover the mystery of the forest paradise that will do anything to not let you leave. This was my senior capstone that I created in a team of 8 over the course of 2 quarters. My main focus for this project was the enemy AI which I implemented using behavior trees...

TacoCat

TacoCat is a 2D story game consisting of truck driving and taco making minigames. I worked on this game as a part of the Game Design and Art Collaboration club which runs a mock game studio every school year to create a game with a team of around 25 people. My main contributions to the project were programming parts of the taco making minigame.

Soaring Hearts

Soaring Hearts is a 2 player Co-op puzzle-platformer made in Unity by a team of 9 based on the prompt "Hidden Tutorials". For this project I worked with a teammate to program a 2D player controller utilizing a finite state machine and implemented all of the player animations/sounds.

  • Play Here
  • Github Repository

Coma Dream

Coma Dream is a 3D puzzle-platformer made in Unity in 2 weeks by a team of 7 based on the prompt "A Room Without Walls". For this project I programmed a 3D player controller using a finite state machine, cuboid planet physics, and AI for the runaway collectible keys.
